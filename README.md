# Desafio Técnico - CronoWise

Solução do desafio técnico para a vaga de Estágio em Desenvolvimento (Processo Seletivo).

## 🚀 Como executar o projeto

1. Certifique-se de ter o **Java 17** (ou superior) instalado na sua máquina.
2. Clone este repositório.
3. Abra o projeto na sua IDE de preferência.
4. Execute a classe principal localizada em: `src/main/java/com/cronowise/Main.java`.
5. Os resultados dos testes práticos (Bloco 1) serão impressos diretamente no console.
